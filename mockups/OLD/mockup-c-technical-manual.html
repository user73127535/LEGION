Section 1

System Overview

LEGION is a system for analyzing the collective performance of player groups in League of Legends. Standard tracking platforms (op.gg, u.gg, etc.) report individual statistics. This system does not.

All metrics computed by LEGION are derived from matches in which two or more members of a registered group (designated "cell") participated simultaneously. This constraint is applied at query time and cannot be overridden.

Note: LEGION requires each user to register a valid Riot ID. Match data is retrieved from the Riot Games API and cached locally after first retrieval.

Section 5

User Registration

New users must complete the following registration procedure:

  1. Provide a valid email address and create a password.
  2. Enter your Riot ID (Game Name and Tag Line).
  3. Submit the form. The system will verify your Riot ID against the Riot Games API.
  4. Upon successful registration, proceed to Authentication (Section 6).
Section 6

Authentication

Returning users authenticate with their registered email and password. Upon successful authentication, the user is directed to the Cell Briefing (Section 2).
